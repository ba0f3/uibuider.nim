# uibuilder.nim
UI Builder for [@nim-lang/ui](https://github.com/nim-lang/ui) using [Glade](https://glade.gnome.org/)


### Whats working?
> "working" means able to display on GUI, nothing else

- [x] Button
- [x] Window
- [x] Box
- [x] Checkbox
- [x] Entry
- [x] Label
- [ ] Tab
- [x] Group
- [x] Spinbox
- [x] Slider
- [x] ProgressBar
- [x] Separator
- [x] Combobox
- [x] EditableCombobox
- [x] RadioButtons
- [ ] MultilineEntry
- [x] MenuItem
- [x] Menu

### Notes
- currently support only one Window
- dont put MenuBar into Window
